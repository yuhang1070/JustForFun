## x 的平方根
### 考察: 字节 1.
### https://leetcode.cn/problems/sqrtx/
```java
class Solution {
    public int mySqrt(int x) {
        double err = 0.000001;
        double l = 0.0, r = 1.0;
        if(x > 1) {
            l = 1.0;
            r = x;
        }
        double m = 0, m2 = 0;
        while(r - l > err) {
            m = (l + r) / 2;
            m2 = m * m;
            if(m2 > x) {
                r = m;
            } else {
                l = m;
            }
        }
        return (int)(r);
    }
}
```
