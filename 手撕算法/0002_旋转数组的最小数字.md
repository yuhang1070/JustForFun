## 旋转数组的最小数字
### 考察: 腾讯 1.
### https://leetcode.cn/problems/xuan-zhuan-shu-zu-de-zui-xiao-shu-zi-lcof/
```java
// 数组中的最后一个元素 xx：在最小值右侧的元素，它们的值一定都小于等于 xx；而在最小值左侧的元素，它们的值一定都大于等于 xx。
// 可以根据这一条性质，通过二分查找的方法找出最小值。
class Solution {
    public int minArray(int[] numbers) {
        int N = numbers.length;
        if(N == 1) {
            return numbers[0];
        }
        if(numbers[0] < numbers[N - 1]) {
            return numbers[0];
        }
        int l = 0, r = N - 1;
        while(l < r) {
            int m = (l + r) / 2;
            if(numbers[m] == numbers[r]) {
                r--;
            } else if(numbers[m] > numbers[r]) {
                l = m + 1;
            } else {
                r = m;
            }
        }
        return numbers[r];
    }
}
```
