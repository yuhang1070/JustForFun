## 数组中重复的数字
### https://leetcode.cn/problems/shu-zu-zhong-zhong-fu-de-shu-zi-lcof/
```java
class Solution {
    public int findRepeatNumber(int[] nums) {
        HashSet<Integer> set = new HashSet<>();
        for(int num : nums) {
            if(set.contains(num)) {
                return num;
            } else {
                set.add(num);
            }
        }
        return 0;
    }
}
```
