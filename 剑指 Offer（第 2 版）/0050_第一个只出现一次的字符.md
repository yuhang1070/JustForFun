## 第一个只出现一次的字符
### https://leetcode.cn/problems/di-yi-ge-zhi-chu-xian-yi-ci-de-zi-fu-lcof/
```java
class Solution {
    public char firstUniqChar(String s) {
        char[] arr = s.toCharArray();
        int[] counter = new int[128];
        for(char ch : arr) {
            counter[ch]++;
        }
        for(char ch : arr) {
            if(counter[ch] == 1) {
                return ch;
            }
        }
        return ' ';
    }
}
```
