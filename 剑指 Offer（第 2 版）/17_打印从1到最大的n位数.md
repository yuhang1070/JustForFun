## 打印从1到最大的n位数
### https://leetcode.cn/problems/da-yin-cong-1dao-zui-da-de-nwei-shu-lcof/
```java
class Solution {
    public int[] printNumbers(int n) {
        n = (int) Math.pow(10, n);
        int[] ans = new int[n - 1];
        for(int i = 0; i < ans.length; i++) {
            ans[i] = i + 1;
        }
        return ans;
    }
}
```
